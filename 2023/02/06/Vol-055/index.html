🧩 苹果 Bento 布局收集

苹果把介绍新产品特性的 keynote 排版风格称之为“Bento”，意为日式便当。Apple Summary Slides 收集了苹果官方出品的 Bento 排版图片。

🧩 当 UI 好到你想触摸它时

🧩 在figma中用Prototype制作的游戏

🧩 World Wide Walls

当你给世界上任何一个人一罐喷漆和一面白墙时，会发生什么？创意工作室 oio 试图用 Chrome 浏览器做一个社会试验，把 Web 砌成 Wall

World Wide Walls 是一个游戏化的 Chrome 插件，它可以将网络页面变成一面全球网友都可以共创的涂鸦墙，任何人都可以在任何网站上留下自己的标记：声援自己喜欢的媒体，增加个性化的注解，或是暂停视频画个 meme…

🧩 Can’t Unsee

Can’t Unsee 便是一款帮助你识别设计细节的游戏，它的玩法非常简单：在两张界面图片中选出你认为更恰当的设计，在得知对错后还可以比较两张图的差异

📦 资源

🧩 专业气象业务通用功能组件

本文基于作者多年的气象行业产品设计经验积累，归纳了气象业务产品中常用的组件类型。

🧩 GPT-3 的使用案例

AI 的时代要来临了，限制我们的不是技术发展，而是产品的想象力。这个网站搜集了非常多 GPT-3 的使用案例，希望能大家都能找到是和自己的使用场景，让 AI 赋能，解放生产力。

🧩 Ul Notes

UI Notes 收集了大量线上优秀 App 的完整 UI 截图，只有落地设计没有飞机稿，你可以在这里探索 UI 设计的最新趋势、浏览竞品的产品设计、快速找到工作中需要的灵感

🧩 矢量可编辑的iOS图标

231个矢量可编辑的iOS图标，作者还在持续更新中

📖 阅读

🧩 Darkroom APP 图标背后的艺术

获得过苹果设计奖的 Darkroom ，分享了他们图标的设计思路：基本图形来自挪威的山脉，从 2.0 版本开始每个版本都基于基本图形做了不同风格的尝试，同时他们也对图标进行横向扩展，把已经确定的图标扩展成 29 个不同视觉效果的变体。

🧩 那些设计巧妙的404页面

理想情况下，用户永远不会找到 404 页面，但错误发生时。巧妙的 404 页面可以娱乐并帮助阻止访问者在找不到他们想要的东西时离开网站。它们甚至可以成为强化品牌标识的有效工具，为可能对品牌有利的错误添加一线希望。

🧩 从自然界中学习设计

除了表象和知识，自然界本身的结构也很有信息量。从视觉设计到用户体验设计，我们可以从自然界中学到很多东西。事实上，达芬奇从生物和自然界中汲取了很多东西，包括黄金比例。

🧩 设计师如何入门前端

无论是设计工具还是前端代码，都只是实现创意的工具。最终能够限制创意的，只有不够开放和好奇的大脑。

🧩 深入了解为界面创建有效按钮背后的设计原则

设计按钮需要考虑风格、反馈和可访问性。设计目标应该是使它们尽可能直观，以便用户可以与按钮无缝交互。本文将介绍按钮类型、用例、描述所有状态并提供一些主题提示。

📰 资讯

🧩 Realme 官宣联名可口可乐

realme 近日在海外宣布，将与可口可乐达成合作，携手推出联名款手机。同时，双方还公布了 realme 吉祥物的全新配色。

🧩 中国电信更新品牌色

最近，中国电信更新了品牌色，目前其官方 App 图标、官方网站均采用全新颜色的 LOGO。全新品牌色由暗沉的深蓝色更替为鲜亮的蓝色，给人一种更年轻更有亲和力的视觉感受。

🧩 国家级出行平台亮相，「强国交通」完成内测即将上线

作为国内首个国家级交通出行平台，涵盖约车、货运、水运、航运等功能的「强国交通」近日已完成内测，进入上线倒计时。
