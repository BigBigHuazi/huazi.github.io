🧩 什么是 Linear 设计风格↗

这是指一种在大面积暗色背景下，使用渐变、模糊、动态流光、极细描边、微噪点、外发光以及庄重的无衬线字体，外加流畅克制的微动效来组织和修饰界面元素的网页设计风格。

🧩 复旦大学 MOSS 正式开源↗

复旦大学自然语言处理实验室开发的新版 MOSS 模型正式上线，成为国内首个插件增强的开源对话语言模型。目前， MOSS 模型已上线开源，相关代码、数据、模型参数已在 Github 和 Hugging Face 等平台开放，供科研人员下载。

🧩 腾讯上线首个「生僻字征集」微信小程序

腾讯征集生僻字的微信小程序上线，通过拍照上传方式，提交无法正常输入的生僻字。除征集生僻字功能外，生僻字征集小程序还设置了科普特色栏目，「生僻字广场」展示最新上传的生僻字，点击可以了解该文字的编码状态、文字释义和来源故事等。
