这里是「BLUE·周刊」的第62期
这周的周刊里充满着动图，好多新出的AI工具，感觉只用图片难以表达。最近在玩 Midjourney 每次生成的图像都出乎意料的惊喜，但最难的还是控制它生成你想要的图片。

🔍 发现

🧩 在黑暗中发光的UI

用摄像头的曝光参数计算出环境光亮度，根据外部光线让 UI 的一部分变暗

🧩 图标微动效

作者 Atilla Taşkıran 用 Spline 创作了许多有趣的动效。

📦 资源

🧩 在figma中插入压缩包

在Figma中添加Zip文件，以便在一个地方拥有所有您需要的文件。视频，文档，字体等，All in figma

🧩 3D 样机插件「Realibox 3D」

即时设计最新的样机工具插件，拥有iPhone Mac 等丰富 3D 样机库，还有海量 3D emoji、icon 等素材。

🧩 阿里设计出品的3D素材网站堆友

堆友是 Alibaba Design 打造的设计师全成长周期服务平台，提供海量素材，目前主要有堆头像和堆场景 2 个功能，3D素材角度、颜色、位置均可在线编辑和实时渲染，另外所有素材也可以单独下载

📖 阅读

🧩 2023奥斯卡电影创意海报汇总

整理了外网上10位海报设计师为今年提名了奥斯卡最佳影片的10部电影设计的艺术海报。其中包含了乐高风、小金人风、剪报风、剪影风、油画风……每张海报上选取的剧情元素都十分经典。

🧩 屏幕之外的设计，聊聊AI和UI设计

🧩 《GPT-4 ，通用人工智能的火花》论文内容精选与翻译

文章节选+翻译了本月最重要的一篇论文的内容，该论文是一篇长达154页的对 GPT-4 的测试。微软在很早期就接触到了 GPT-4 的非多模态版本，并进行了详尽的测试。论文不管是测试方法还是测试结论都非常精彩，强烈推荐。

📰 资讯

🧩 Adobe 发布 Firefly，加入 AIGC AI 绘图战场

🧩 GitHub 发布了 Copilot X

GitHub Copilot + GPT-4 = COPILOT X，引入了 GPT-4，并且还支持了聊天和语音。

🧩 ChatGPT 开始支持第三方插件

“ChatGPT宇宙”开始了，有了这套第三方插件系统，现有产品能更容易地接入 ChatGPT，它们的专业能力和 ChatGPT 的自然语言人机交互、通用知识库结合，将把 ChatGPT 的能力提升到新层面。

🧩 微软正式发布 Notion 竞争产品：Loop

Loop 不但能实现 Notion 擅长的 all-in-one 式编辑和协作体验，还把word、excel 文件整合到一起。

🧩 Figma 实现了跨设备拷贝-粘贴

现在可以跨桌面和 iPad 在 Figma 和 FigJam 文件之间复制对象、框架、文本、小部件等。

🧩 U Design Week 2023定档6月1日

U设计周（U Design Week）2023定档6月1日，于杭州国际博览中心正式开幕，主会场活动为期4天，推出全新主题“时代的设计Designed by U”，也将打造更多精彩无限的升级内容。

🧩 OPPO 更新 LOGO

OPPO邀请了斯德哥尔摩设计实验室（SDL），进行品牌升级。最大的变化就是标准色变成了更加鲜亮的绿色。这个更加鲜亮的绿色，带来了视觉刺激，同时，也让OPPO这个品牌显得更加年轻活力。
